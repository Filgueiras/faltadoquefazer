Eu estava indo dormir quando lembrei do modo de visualizacao do desenvolvedor, no Internet Explorer 9...
Bom, descobri o problema que fazia o script nao executar neste navegador: usei uma CONSTANTE para o preco do bolo, R$ 18,00. Estava tudo certo com os navegadores da concorrência, mas o erro no IE caiu bem na linha de declaração da CONSTANTE. Aí lembrei do que o livro de JavaScript dizia, sobre nem todos os navegadores aceitarem este comando. Teste rapido: troquei o comando CONST por var e voi lá, funcionou !! ;-)
